Nicolas Fonseca Sequera

Perfil

Tengo conocimiento basico en HTML, CSS y JavaScript con los cuales soy capaz de crear una interfaz agradable, precisa e interactiva al gusto del usuario. Tambien tengo habilidades de diseño visual y experiencia basica con analisis de percepción del usuario

Temas

Para la realización de este proyecto tuve en cuenta los elementos ofrecidos por el framework de bootstrap para algunas cosas basicas del sitio como las card, el menu de navegación, feauters, etc. Sin embargo la mayoria de los elementos presentados fueron creados por medio de la investigación y combinación de diferentes codigos y estilos tanto del framwork como de internet directamente.
En su mayoria el sitio web to,a una tematica neutra con colores solidos como el verde, purpura y balnco para hacer contrastes agradables a la vista y no capciones, ademas, fue diseñado con poco texto pensando en el cansancio que puede tener el usuario al tener gran variedad de texto. Otra cosa importante que tiene el sitio es el uso del local-storage para el almacenamiento de algunas funciones como los son: "servicios", "empleos" y "opiniones".

Experiencia

La experiencia de crear una pagina responsive desde cero fue agradable ya que pude obtener mas información de la que ya tenia, ademas de pemitirme explorar diferentes estilos que pueden ser utilizados en el sitio. Aunque he de admitir que la tematica de animación de eventos sale de mi zona de confort totalmente pero con ayuda de algunos sitios de internet logre obtener conocimiento para crear este sitio.
